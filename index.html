    How was this made? 🤓

    ×

    This database wasn't typed by hand! All deck lists were extracted automatically using computer vision, written in Python.

    • The Cards: Scraped directly from YouTube video screenshots by isolating the card slots and reading the counts.
    • The Names: Grabbed by applying color-masking to the deck title banners.
    • The Videos: Matched directly to FryEmUp's YouTube channels using script automation to pull the links and titles.
    • The Images: Pulled from the PvZ Fandom Wiki infoboxes.
    • The Card Data: To make all the beautiful stats work, the base game data was sourced from this repo and modified into JSON files.

    Everything was then compiled and cleaned up! You can download the raw data for yourself below. Please give credit if you use it for any commercial purposes.
