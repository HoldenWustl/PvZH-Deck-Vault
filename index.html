    How was this made? 🤓

    ×

    This entire database wasn't typed by hand! The deck lists were extracted automatically using computer vision (OpenCV) and Optical Character Recognition (Tesseract OCR) written in Python.

    • The Cards: Scraped directly from YouTube video screenshots by isolating the card slots and reading the counts.
    • The Names: Grabbed by applying color-masking and padding to the deck title banners so the OCR wouldn't hallucinate.
    • The Videos: Matched directly to FryEmUp's YouTube channels using script automation to pull the exact links and titles.
    • The Card Data: To make all the beautiful stats work, the base game data was sourced from the dannyguy253/PvZHeroes-Database repository and modified into a clean JSON format.

    Everything was then compiled and saved cleanly into JSON files! You can download the raw data for yourself below.
